If the user invokes this skill without any other guidance, ask them what they want to build or design, ask some questions, and act as an expert designer who outputs HTML artifacts _or_ production code, depending on the need.

## Quick reference
- **Brand:** Swift7 — affordable web design for UK small businesses. £500 all-in, live in 7 days.
- **Foundations:** `colors_and_type.css` (all `--s7-*` tokens + semantic type classes).
- **Voice:** plain-spoken sole-trader confidence; British English; sells on speed, fixed price, leads; no emoji; lime `→`/`✓` as markers. See README "CONTENT FUNDAMENTALS".
- **Look:** dark forest-green canvases + one electric lime accent (`#A8C94A`), Poppins (900 for headlines, tight tracking), 4px radius, soft dark shadows, calm one-easing motion. See README "VISUAL FOUNDATIONS".
- **Icons:** Lucide line icons (2px stroke, currentColor) + filled WhatsApp/Google brand glyphs. See README "ICONOGRAPHY".
- **Assets:** `assets/` (logo, app-icon, founder photo, client mockups).
- **UI kit:** `ui_kits/website/` — modular JSX recreation of the marketing site; lift buttons, nav, cards, the lead modal.
- **Specimens:** `preview/` — small HTML cards showing each token/component in use.

⚠️ The `client/` StrangeSkies app in the original codebase is a *separate* brand (purple/void) — do not use its palette for Swift7 work.
